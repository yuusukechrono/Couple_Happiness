# カプハピ（カップルハピネス）

# 概要
より良い関係になりたいと思っているカップルの
普段は恥ずかしくて、相手の反応が恐くて言えない考えを
フリック・クリック１つで届けてコミュニケーションのきっかけを生む
それがカップルハピネス！

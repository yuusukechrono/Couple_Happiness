# カプハピ（カップルハピネス）

# 概要
より良い関係になりたいと思っているカップルの<br>
普段は恥ずかしくて、相手の反応が恐くて言えない考えを<br>
フリック・クリック１つで届けてコミュニケーションのきっかけを生む<br>
それがカップルハピネス！
